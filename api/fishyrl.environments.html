class fishyrl.environments.VectorizedRLGymEnvironment(env_name: str = 'Soccar', num_envs: int = 1, team_size: int | list[int] = 2, frame_skip: int = 8, allow_rendering: bool = False, automatic_reset: bool = True, **init_kwargs: dict[str, Any])#

Bases: VectorizedEnvironment

A vectorized environment wrapper for RLGym environments.

__init__(env_name: str = 'Soccar', num_envs: int = 1, team_size: int | list[int] = 2, frame_skip: int = 8, allow_rendering: bool = False, automatic_reset: bool = True, **init_kwargs: dict[str, Any]) → None#

Initialize the vectorized RLGym environment.

Parameters:

  • env_name (str) – The name of the RLGym environment to create, currently unused. (Default: 'Soccar')

  • num_envs (int) – The number of parallel environments to create. (Default: 1)

  • team_size (int | list[int]) – The size of both teams. Can be an int or a list of ints for each environment. (Default: 2)

  • allow_rendering (bool) – Whether to allow rendering of the environment, frame grabbing is currently unsupported in favor of live visualization. (Default: False)

  • frame_skip (int) – The number of frames to skip for each action. (Default: 8)

  • automatic_reset (bool) – Whether to automatically reset the environment after termination. (Default: True)

  • init_kwargs (dict[str, Any]) – Additional keyword arguments for initializing the RLGym environment. (Default: {})

render(delay: bool = True, speedup: float = 1.0, warn: bool = True) → ndarray#

Render the environments and return a frame.

Live visualization is currently supported, but frame grabbing is not due no support with RLViser. This method will render the first environment to the RLViser window, but will return an empty array.

Parameters:

  • delay (bool) – Whether to delay rendering to match the render FPS. (Default: True)

  • speedup (float) – A factor to speed up rendering, applied to the delay. (Default: 1.0)

  • warn (bool) – Whether to display warnings on long computations. (Default: True)

Returns:

An array of rendered frames, with shape (num_envs, height, width, channels).

Return type:

np.ndarray
